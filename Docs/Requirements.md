# Requirements

### Authorization:

* Own registration 
* OAuth

### Photos:

* Categories
* Tags
* Views
* Likes
* Comments
* Exif
* Geotags

### Users:

* Personal data 
	* Name, Surname
	* Website, Facebook/Twitter profile
	* Avatar
	* Location
	* Mobile phone
* Equipment (+ "Potato" and "TAPOK-M")
* Followers
* Rating
* Best rated photo
